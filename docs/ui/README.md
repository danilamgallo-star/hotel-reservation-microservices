## Cómo generarlas

Usé [Uizard](https://uizard.io) para hacer los mockups rápido. Básicamente:

1. Te registras gratis
2. Creas un proyecto nuevo con IA
3. Le das los prompts que están en el README principal
4. Exportas las imágenes y las pones aquí

Nada del otro mundo, pero funciona bien para prototipar
